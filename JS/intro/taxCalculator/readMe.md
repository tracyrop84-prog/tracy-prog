# TAX CALCULATOR

## Steps to follow in calculating tax are:

1.  determine the gross salary
2.  calculate the statutory deductions ie: shif,nssf
    *Shif apply 2.75% of gross salary
    *nssf use the tiered pension rate to th gross salary
    |tire I| 6% of first kes 9000 |
    |tire II|6% 0f KES 9001-108,000|
3.  find the taxable income`taxableIncome=grossSalary-shif-Nssf`
4.  calculate paye and apply personal relief
    \*for paye use the pay tax bands
5.  calculate the net salary`netSalary=grossSalary-paye-shif-nssf`
